# 特定商取引法に基づく表記(ひな形)

> 本ひな形は一般的な参考情報であり、法的助言ではない。実際の掲載前に専門家(弁護士・行政書士等)の
> 確認を推奨する。無料の情報提供・診断のみを行う段階では特定商取引法の表記義務は生じないが、
> 有料プラン等の提供を開始する際には整備が必要となる。

| 項目 | 内容 |
|---|---|
| 販売事業者 | [要記入: 氏名または屋号] |
| 運営統括責任者 | [要記入: 氏名] |
| 所在地 | [要記入。個人事業主は原則自宅住所の開示が必要だが、消費者庁の考え方に基づき一定の要件を満たす場合、請求があれば遅滞なく開示する旨を明示した上でバーチャルオフィスの住所等の代替表示が認められる場合がある。要専門家確認] |
| 連絡先 | [要記入: 電話番号・メールアドレス。個人の場合、請求に応じ遅滞なく開示する旨の記載で代替できる場合がある] |
| 販売価格 | [要記入: 有料プランの価格。消費税込み表示] |
| 商品代金以外の必要料金 | [要記入: 通信費等、利用者負担が発生するものがあれば記載] |
| 支払方法 | [要記入: クレジットカード等] |
| 支払時期 | [要記入] |
| サービス提供時期 | [要記入: 決済完了後即時 等] |
| 返品・キャンセルについて | [要記入: デジタルサービスの性質上返金不可とする場合はその旨を明記] |
| 動作環境 | [要記入: 推奨ブラウザ等] |

## 注記

- 本サービスは「セルフヘルプ支援ツール」であり、司法書士・税理士としての有償独占業務(登記申請書類の
  作成代理、税務書類の作成代理)は行わない旨をあわせて明記することを推奨する
  ([法的リスク](legal-risks.md)参照)
