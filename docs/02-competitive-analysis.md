## 2. 同一性(パリティ機能) — これがないと土俵に立てない

競合の多くが共通して提供している、**利用者が最低限期待する機能**:

1. 必要書類チェックリストの自動生成(相続人関係・不動産有無等の入力に応じて変動)
2. 相続関係説明図・遺産分割協議書などの**書式ひな形の自動作成**
3. 法務局への申請書類一式の出力(印刷可能なPDF等)
4. 進捗状況の可視化(どこまで終わったか)
5. 料金・所要期間の明確な提示

これらは本サービスでも「セルフ登記ガイド」機能として必ず実装する([要件定義](05-site-design-requirements.md)参照)。

## 3. 優位性(差別化ポイント) — ここで勝つ

競合調査から見えた**既存サービスに欠けている/弱いポイント**:

1. **「自分でできるか / 代行すべきか」を最初に判定する診断ロジックがない**
   - そうぞくドットコムは最初から代行前提、better相続はセルフ前提で、いずれも「あなたにはどちらが
     向いているか」を診断する導線を持たない
   - 本サービスは診断を起点に置き、**セルフ推奨・代行推奨・ハイブリッド(一部だけ代行)**の3パターンで
     結果を出す設計にすることで、ユーザーの意思決定を支援する独自ポジションを取れる
2. **AIチャットによる個別質問対応がない**
   - 既存サービスはFAQページや電話サポートが中心。AIチャットで「うちのケースはどうすればいい？」に
     即答できる体験は差別化になる(ただし[法的リスク](08-legal/legal-risks.md)に注意し、個別の法律相談
     ではなく一般的な情報提供に留める設計とする)
3. **有料広告に依存しない集客(SNSバイラル)**
   - そうぞくドットコムはCPC広告、better相続は銀行提携という、個人副業では再現できないチャネルに依存
   - 本サービスは「相続あるある」「義務化の期限は？」等の認知コンテンツをSNS(X/Instagram/TikTok)で
     発信し、無料ツールとしての口コミ・シェアを起点にした集客を軸にする
4. **セルフと代行のハイブリッドをシームレスに提供**
   - 診断結果に応じて、セルフ対応部分は無料ツールで完結させつつ、複雑な部分(遺産分割で揉めている、
     相続人が多数・海外在住など)だけを提携専門家に送客する「部分代行」導線を持つ競合は見当たらない

## 4. ポジショニングまとめ

```
        セルフ度が高い ←――――――――――――――――――――――→ 代行度が高い
better相続登記  相続登記ラボ系          [本サービス]           そうぞくドットコム
(セルフ特化)   (セルフ特化・価格競争)  (診断で最適解を提示、  (代行特化・広告依存)
                                        ハイブリッド対応)
```

本サービスは「診断によって最適な進め方(セルフ/代行/ハイブリッド)を提示する」ポジションを取ることで、
セルフ専業・代行専業のどちらとも直接競合せず、**両方の入口になる**ことを狙う。
