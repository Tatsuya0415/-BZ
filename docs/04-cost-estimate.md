# コスト試算(副業運営を前提)

前提: 個人が本業と並行して副業として開発・運営する。外注は最小限、自分で開発する想定。
金額は2026年7月時点の一般的な相場感からの概算であり、実際の契約内容により変動する。

## 1. 初期費用(概算)

| 項目 | 概算費用 | 備考 |
|---|---|---|
| ドメイン取得 | 年1,000〜3,000円 | `.com`等 |
| ロゴ・ブランディング | 0〜5万円 | 自作 or ココナラ等での外注 |
| 商標調査(任意) | 0〜3万円 | サービス名を本格展開するなら推奨 |
| 開発(自作前提) | 0円(自分の工数) | Figmaでの設計→コード実装は自分で行う想定 |

## 2. 月額ランニングコスト(概算)

| 項目 | 概算費用/月 | 備考 |
|---|---|---|
| ホスティング(Vercel/Netlify等) | 0〜3,000円 | 個人規模なら無料枠で足りるケースが多い |
| ドメイン費按分 | 100〜300円 | 年額を按分 |
| AI API利用料(診断ロジック・チャット) | 3,000〜3万円 | 利用量・モデルにより大きく変動。診断ロジック自体はルールベースにしてAIはチャット相談等の補助用途に限定すればコストを抑えられる |
| メール送信サービス(問い合わせ等) | 0〜1,000円 | 小規模なら無料枠で十分 |
| アクセス解析(GA4等) | 0円 | 無料 |
| 画像・アイコン素材 | 0〜1,000円 | フリー素材中心なら実質0円 |
| **合計目安** | **月5,000〜4万円程度** | フェーズ1(収益化前)は低コストに抑える設計が可能 |

## 3. フェーズ2以降で発生しうるコスト

| 項目 | 概算費用 | 備考 |
|---|---|---|
| 提携司法書士・税理士の開拓 | 基本0円 | 着手金なし・成約時マージンの送客契約であれば初期費用は発生しにくい |
| 弁護士・司法書士への法務レビュー(推奨) | 5〜20万円(スポット) | [法的リスク](08-legal/legal-risks.md)に記載の非弁・非司法書士・非税理士行為の該非確認のため、サービス公開前に一度は専門家レビューを受けることを強く推奨 |
| 特定商取引法・プライバシーポリシー整備 | 0円(ひな形使用)〜数万円(専門家確認) | [法務ひな形](08-legal/)参照 |
| 有料プラン決済導入(Stripe等) | 決済手数料3.6%前後 | フェーズ2で有料プランを始める場合 |

## 4. 時間コスト(金銭以外で最も重要)

副業運営における最大のコストは**時間**である:

- サイト開発・保守: 週数時間〜(自作前提)
- SEOコラム記事の執筆: 週1〜3本ペースが望ましいが、副業では現実的なペース配分が必要
- SNS運用(X/Instagram/TikTok): 週数本の投稿・コメント対応
- AIを執筆・診断ロジックの補助に活用することで、コンテンツ制作の時間コストを圧縮できる
  ([マーケティングプラン](07-marketing-plan.md)参照)

## 5. まとめ

- フェーズ1(集客優先・収益化前)は**月数千円〜数万円程度**の低コストで運営可能
- 最大の投資対象は「時間」であり、AI活用によるコンテンツ・診断ロジック構築の効率化が鍵
- フェーズ2移行時は、法務レビュー費用(スポットで数万〜20万円程度)を事前に見込んでおくことを推奨
