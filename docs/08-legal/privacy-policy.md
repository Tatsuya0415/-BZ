# プライバシーポリシー(ひな形)

> 本ひな形は一般的な参考情報であり、法的助言ではない。実際の掲載前に弁護士等の専門家確認を推奨する。

[事業者名/屋号]([要記入]、以下「当方」という)は、本サービス(以下「本サービス」という)における
利用者の個人情報の取扱いについて、以下のとおりプライバシーポリシー(以下「本ポリシー」という)を定めます。

## 1. 取得する情報

本サービスでは、診断・シミュレーション・書式作成機能の提供にあたり、以下の情報を取得する場合があります。

- 氏名、住所、生年月日等の相続人・被相続人に関する情報
- 財産構成に関する情報(不動産の有無、財産のおおよその金額帯等)
- メールアドレス等の連絡先情報(専門家送客を希望した場合)
- アクセスログ、Cookie等の技術情報

## 2. 利用目的

取得した情報は以下の目的で利用します。

1. 診断結果・シミュレーション結果の提供
2. 書式ひな形(相続関係説明図、遺産分割協議書等)の作成支援
3. 利用者が希望した場合の、提携先(司法書士・税理士等)への紹介
4. サービス改善のための統計的分析(氏名等の特定個人情報を除き、匿名化した形で利用)
5. お問い合わせへの対応

## 3. 第三者提供

以下の場合を除き、取得した個人情報を本人の同意なく第三者に提供することはありません。

- 利用者が専門家紹介を希望し、提携先の司法書士・税理士等へ連絡先情報を提供する場合(事前に利用者の
  同意を得た上で行う)
- 法令に基づく場合

## 4. 安全管理措置

- 通信の暗号化(TLS)を実施します
- 診断・シミュレーションで入力された情報のうち、進捗保存機能を利用しない場合はセッション終了後に
  破棄します
- アクセス権限の管理等、個人情報への不正アクセス防止のための措置を講じます
