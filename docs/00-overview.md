# 相続手続きセルフ支援サービス — 全体サマリ

## サービスコンセプト

**「自分で相続登記・相続税申告を進めたい人」のためのセルフヘルプ・プラットフォーム。**
状況を診断し、自分で対応できる範囲は無料の書式・チェックリスト・進捗管理ツールで支援し、
難易度が高い/自分では対応しきれないケースは提携司法書士・税理士への送客につなぐ。

- 主軸: **セルフ登記支援**(相続登記の書式自動作成・必要書類チェックリスト・進捗管理)
- 副軸: 相続税申告シミュレーター(自己申告可能な簡易ケースの見極め)、複雑案件向けの**代行専門家紹介**
- 運営形態: 個人の副業としてスタート。収益化は後回しにし、まずAI活用・SNS活用でユーザー数を増やす
- ゴール: 相続セルフ支援カテゴリで**利用者数No.1**

## 意思決定の要点(結論だけ先に)

1. **市場は伸びている**: 2024年4月の相続登記義務化以降、登記件数は前年比8.8%増(約120万件/年)。
   義務化の認知度は7割を超えたが、期限まで正確に知る人は4割強に留まり、**セルフガイドの需要余地は大きい**
   ([出典](01-market-research.md))
2. **「広告費のみで収益化」は単体では弱い**: 競合(そうぞくドットコム、better相続、つぐなび等)は
   いずれも定額課金型 or 送客手数料型で収益化しており、純広告表示モデルで成立している競合は見当たらない。
   ユーザーの構想通り「集客優先→収益化は後回し」で進めるのは妥当だが、将来の収益化は**送客手数料+有料プラン**
   とのハイブリッドを前提にすべき ([詳細](03-business-model.md))
3. **最大の注意点は法律業務との境界線**: 登記書類の有償代行は司法書士の、申告書の有償作成代理は税理士の
   独占業務。本サービスは「代行」ではなく「セルフヘルプ支援(書式ひな形・シミュレーション・チェックリスト)」
   に徹し、代行が必要なケースは資格者への送客に限定する設計とする ([詳細](08-legal/legal-risks.md))
4. **差別化の核**: 既存競合は「セルフ or 代行」のどちらかに偏っており、**自分でできるか/代行すべきかを
   最初に判定する診断ロジック**を持つサービスがない。ここを本サービスの独自ポジションとする
   ([詳細](02-competitive-analysis.md))
